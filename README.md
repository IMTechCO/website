# IM Tech

> Transformamos tecnología en resultados.

Sitio web corporativo oficial de IM Tech, desarrollado con tecnologías modernas para ofrecer una experiencia rápida, accesible y optimizada para SEO.

---

## Tecnologías

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- React Hook Form
- Zod
- Resend

---

## Características

- Landing corporativa
- Diseño responsive
- SEO optimizado
- Open Graph
- JSON-LD
- Sitemap automático
- Robots.txt automático
- Formulario de contacto
- Integración con Resend
- Animaciones fluidas
- Botón de WhatsApp

---

## Instalación

```bash
npm install
npm run dev
```

---

## Build

```bash
npm run build
```

---

## Variables de entorno

Crear un archivo `.env.local` con:

```env
RESEND_API_KEY=
CONTACT_EMAIL=
NEXT_PUBLIC_WHATSAPP_NUMBER=
```

---

## Producción

```bash
npm run build
npm start
```
